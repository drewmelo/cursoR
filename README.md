Este repositório reúne os materiais do curso **Introdução à análise de dados com R aplicado à Economia**, com início em **10 de abril de 2026**.

O curso foi estruturado para apresentar, de forma prática e progressiva, o uso do R na análise de dados econômicos, desde os fundamentos até aplicações com visualização e desenvolvimento de projetos.

## Estrutura do repositório

Os materiais estão organizados da seguinte forma:

```
📦 cursoR
 ┣ 📜 aula01.R
 ┣ 📜 aula02.R
 ┣ 📜 aula03.R
 ┣ ...
 ┣ 📂 slides
 ┃ ┣ 📄 aula01.pdf
 ┃ ┣ 📄 aula02.pdf
 ┃ ┣ 📄 aula03.pdf
 ┃ ┗ ...
```

* **Arquivos `.R`**: scripts utilizados em aula, organizados por número (aula01, aula02, etc.)
* **Pasta `slides/`**: apresentações em PDF utilizadas durante as aulas

## Objetivo do curso

O curso tem como objetivo desenvolver habilidades práticas em análise de dados com R, com foco em aplicações econômicas, incluindo:

* Manipulação e organização de dados
* Criação de indicadores
* Análise exploratória
* Visualização de dados
* Construção de narrativas com dados

## Ementa do curso

### Módulo 1 - Fundamentos do R e pensamento com dados

* Introdução ao R, operações básicas e criação de objetos
* Estruturas de dados e funções básicas
* Importação e exportação de dados
* Estruturas condicionais, de repetição e loops

### Módulo 2 - Manipulação de dados com tidyverse

* Exploração e seleção de dados com dplyr
* Transformação de variáveis e criação de indicadores
* Agregação de dados e integração de bases (joins)
* Limpeza, organização e reestruturação de dados

### Módulo 3 - Visualização de dados com ggplot2

* Fundamentos do ggplot2 e construção de gráficos
* Gráficos básicos aplicados à análise econômica
* Personalização e refinamento de visualizações
* Visualização de dados e storytelling

### Módulo 4 - Aplicações em Economia e desenvolvimento de projeto final

* Estatística descritiva e análise exploratória aplicada a dados econômicos
* Introdução à reprodutibilidade com Quarto
* Desenvolvimento e apresentação de projeto final

## Metodologia

O curso segue uma abordagem prática, baseada em:

* exemplos aplicados à economia
* construção progressiva de conhecimento
* exercícios ao final de cada aula
* uso de dados reais sempre que possível

## Requisitos

Para acompanhar o curso, recomenda-se:

* R instalado
* RStudio instalado

Pacotes utilizados ao longo do curso:

```r
install.packages("tidyverse")
```

## Como utilizar este repositório

1. Clone o repositório:

```bash
git clone https://github.com/drewmelo/cursoR.git
```

2. Abra os scripts `.R` no RStudio

3. Execute os códigos conforme as aulas

4. Utilize os slides como apoio teórico

## Observações

* Os scripts foram desenvolvidos para fins didáticos
* Alguns exemplos utilizam dados simulados para facilitar o aprendizado
* Recomenda-se executar os códigos passo a passo

## Sobre o curso

Este curso foi desenvolvido com foco em estudantes de Economia interessados em aplicar técnicas de análise de dados em problemas reais, utilizando o R como ferramenta principal.
